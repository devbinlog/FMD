# FMD (Find My Design)

## Overview
FMD is a web application that converts user input (text prompt or canvas sketch) into a structured design profile and finds similar purchasable products across providers.

No authentication.
No payment processing.
Outbound purchase links only.

## Core Flow
1. User enters text or draws on canvas.
2. System analyzes input and generates DesignProfile.
3. Search runs across providers.
4. Ranking engine orders results.
5. User receives product list with explanations.

## Goal (MVP)
- End-to-end working pipeline
- 1 provider integration
- Ranking v1
- Clean minimal UI
- Free-tier deployment

## Non-Goals (MVP)
- Checkout
- User accounts
- Advanced ML training
- Real-time personalization
